Command-Line Interface

  1. bash is a text based interperter that provides a cli for controlling your computer or operating system
    • Bourne-Again SHell
    • display a list of files inside the active folder(ls)
    • detailed listing(ls -l)
    • sort by file time(ls -t)
    • sort by file size(ls -S)
    • reverse the sorting(ls -lSr or ls -ltr)
    • see hidden files(ls -a)
    • current folder(.)
    • parent folder(..)
  2. change directories(cd)
  3. previous directory(cd -)
  4. create folder(mkdir <foldername>
  5. remove folder(rmdir <foldername>
    • Note: all files in the folder must be deleted before you can delete the folder.
  6. create empty file(touch <filename>)
  7. move or rename files(mv)
    • move file(mv filename ./path)
    • rename program(mv original_program.rb Renamed_program.rb)
    • you can also combine the two(mv temp_download.gif ~/Desktop/cats_with_weapons/ninja_cat.gif)
  8. copy files(cp)
    • same syntax as (mv)
    • If you want to copy a directory and its file contents, you need to use the -r flag(cp -r february_cat_gifs ~/Desktop/vital_media_files)
  9. remove file(rm <filename>)
    • remove all files in current directory(rm *)
    • to delete a directory the (-r) flag is needed
    • aditional removal flags
      1. -i (interactive): Prompts you to confirm the deletion. This option can stop you from deleting a file by mistake.
      2. -f (force): Overrides interactive mode and removes the file(s) without prompting. Use this with caution. This action cannot be undone.
      3. -v (verbose): Shows the progress of the files as they are being removed.
  10. edit plain text file(nano /path/to/file)
  11. display files(cat)
    • Note: most likely too big
  12. more or less commands(more <filename> or less <filename>)
  13. logged in username(whoami)
  14. current working directory(pwd)
  15. home directory(cd ~)
  16. (explorer.exe my_file.html)Opens html files in ubuntu in chrome
